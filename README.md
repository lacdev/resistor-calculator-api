## Installation

```bash
$ npm install
```

## Running the app

```bash
# development on PORT 8000
$ npm run start

# watch mode on PORT 8000
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
